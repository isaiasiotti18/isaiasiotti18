# 👨‍💻 Sobre mim

Opa! Bem-vindo ao meu GitHub.
Sou desenvolvedor fullstack com experiência sólida no ecossistema JavaScript e atuação tanto em frontend quanto em backend. Para mim, linguagens e frameworks são ferramentas — e o que realmente importa é entregar soluções escaláveis, performáticas e bem arquitetadas.

Tenho solidos conhecimentos em HTML5, CSS3 e JavaScript (ES6+), além de trabalhar com Node.js e React.js/React Native. Também atuo com C# e ASP.NET no desenvolvimento de aplicações robustas. Minha stack inclui ainda Docker, Linux, AWS (EC2, S3, Lambda, etc.), além de práticas de DevOps, automação com Shell Script e versionamento de código com Git/GitHub.

Trabalho com boas práticas de desenvolvimento, aplicando Clean Code, SOLID, TDD, integração contínua (CI/CD) e design patterns sempre que possível. Acredito em arquiteturas modernas como microservices, serverless e cloud-native, buscando sempre resiliência, segurança e escalabilidade.

No frontend, priorizo componentização, responsividade e performance; no backend, foco em APIs REST, GraphQL, autenticação JWT, controle de sessão, segurança e otimização de queries em bancos de dados relacionais e não relacionais.

Minha filosofia é simples: código bem escrito gera menos dívida técnica e mais valor para o negócio. Gosto de atuar em times ágeis, promovendo colaboração, code review de qualidade, pair programming e cultura DevOps.
